# PersonalSiteAPI

## Description
PersonalSiteAPI is a comprehensive Node.js API designed to manage and serve content for a personal website. It features dedicated endpoints for home, about, education, experience, portfolio, and contact pages.

## Technologies Used
- Node.js
- Mongoose
- MongoDB
- Render

## Endpoints
1. `/home`: Serves the homepage content.
2. `/about`: Serves the about page content.
3. `/education`: Serves the education page content.
4. `/experience`: Serves the experience page content.
5. `/portfolio`: Serves the portfolio page content.
6. `/contact`: Serves the contact page content.
